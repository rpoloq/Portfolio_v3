Virtual Reality

Scape Room

Este proyecto fue desarrollado durante mis estudios de Master. Esta apliación consiste en un juego simple realizado para poner en prácticas las mecánicas de movimeiento en VR alternativas a las habituales usando joystick o teletransportacion. El videojuego consiste emula una scape room donde se debe escapar de una torre trepando, a través de las rocas que sobresalen. la dinámica consiste en encontrar la rotación correcta del "cilindro" superior donde están las rocas del nivel superior e ir subiendo antes de que que te alcance la lava que va ascendiendo para acabar el juego.

Se guarda un registro del mejor tiempo en elq ue se ha conseguido completar el desafío.

This project was developed during my Master's studies. This application is a simple game created to implement alternative VR movement mechanics to the usual joystick or teleportation methods. The game simulates an escape room where you must climb a tower by navigating through protruding rocks. The dynamics involve finding the correct rotation of the upper "cylinder" where the rocks of the higher level are located and climbing up before the rising lava catches up with you, ending the game.

A record of the best time achieved in completing the challenge is kept.

Augmented Reality

Interactive Objects Application

Este proyecto fue desarollado durante mis estudios de Master de nuevo. Consiste en una aplicación hecha en Unity haciendo uso de ARFundation. Es una aplicación en la que se lleva a cabo detección de plano para la colocacción de los objetos virtuales, con los cuales se interactua a través de los hotspots que aparecen sobre el objeto. La apliacación posee una lógica de control de estados en los que está para realizar el proceso de instanciación e interacción con los objetos.

This project was developed again during my Master's studies. It is an application made in Unity using ARFoundation. The application involves plane detection for the placement of virtual objects, which are interacted with through hotspots that appear on the object. The application has a state control logic to facilitate the instantiation and interaction process with the objects.

AR Touristic Application

Aplicación realizada con Unity durante los estudios de curato curso de Ingeniería Multimedia desarrollada con Vuforia en AR. En ella se pueden realizar diferentes interaccions con objetos virtuales sobre un mapa fisico de la ciudad de valencia, jugar a miniuegos e interactuar por voz.

Application developed with Unity during the last year of Multimedia Engineering studies, using Vuforia for AR. It allows various interactions with virtual objects on a physical map of the city of Valencia, playing mini-games, and interacting through voice commands.
